### 문자
1. 배열에 문자 저장하기
```python
>>> str1 = 'python'
```

2. 인덱싱
```python
>>> str1[0]
'p'
```

3. 슬라이싱
```python
>>> str1[0:2]
'py'
>>> str1[ :2]
'py'
>>> str1[2:]
'thon'
```

4. meta-character & escape chracter
- 줄바꿈 `\n`
- tab `\t`

- 줄바꿈 적용 안하게 `\\n`

5. 문자 조작 함수
- 대문자 변환 `upper()`
- 소문자 변환 `lower()`
- 문자 변환 `replace()`
- 문자열 분리 `split()`
- 길이 확인 `len()`
