1. 파이썬 변수 저장하기
```python
>> var1 = 10
>> var2 = 20
```

2. 파이썬 변수 값 출력하기
```python
>> var1  // 10
```

3. 데이터 출력하기
```python
>> print('example %d' %var1)
// %d - int / %f - float / %s - string / %c - character
```

4. 사용자에게 값 입력받기  
`input()` 함수는 어떤 값이든 스트링으로 저장한다

```python
>> no1 = input()

>> no2 = input('please input number:')
please input number:
```

숫자로 입력받기
```python
>> no1 = int(input())
```
